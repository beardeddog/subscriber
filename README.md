subscriber
==========

A simple golang stomp subscriber


Depends
=======

Depends on the following packages

  `go get github.com/gmallard/stompngo`

  `go get github.com/ogier/pflag`

Usage
=====
  `./subscriber -h`
  
  `./subscriber -b broker.net.com:61616 -q TEST  --username=admin --password=admin`
